## Getting Started in Aurora Focus

Follow this sequence when you are new to scripting and deployment in Aurora Focus:

1. Learn the guide layout and conventions.
   - Read [JavaScript Developers Guide](/docs/javascript-developers-guide), [About this Guide](./t-aurora-js-about-this-guide), [Notational Conventions](./r-notational-conventions), and [Icon Conventions](./r-icon-conventions).
2. Confirm the runtime and editor context.
   - Read [Using the JavaScript Environment](./c-aurora-js-environment), [Using the JavaScript Editor](./t-aurora-js-using-the-javascript-editor), and [Using the Editor for a Deployed Job](./t-aurora-js-using-the-editor-for-a-deployed-job).
3. Understand execution behavior before writing logic.
   - Read [Operation Mode](./t-aurora-js-operation-mode), [Program Structure](./c-aurora-js-program-structure), [Script Invocation](./t-aurora-js-script-invocation), [Example Script Lifecycle](./c-aurora-js-example-script-lifecycle), and [Managing Execution Time](./t-aurora-js-managing-execution-time).
4. Build your first script around data flow.
   - Read [Input Data](./t-aurora-js-input-data), [Constant Data](./t-aurora-js-constant-data), [Output Data](./t-aurora-js-output-data), [Job Status](./t-aurora-scripting-job-status), and [Text Encoding](./c-aurora-js-text-encoding).
5. Add device and interface controls as needed.
   - Read [TCPIP, RS232, and USB CDC Serial](./t-aurora-scripting-tcpip-rs323-and-usb-cdc-serial), [HID Keyboard](./t-aurora-js-hid-keyboard), [FTP Control](./t-aurora-js-ftp-control), [GPIO Ports](./t-aurora-focus-js-gpio-ports), and [Beeper](./t-aurora-focus-js-beeper).
6. Validate and harden your implementation.
   - Read [Debugging](./c-aurora-js-debugging), [Using Simulated Data](./t-aurora-js-using-simulated-data), [Using Custom Templates](./t-aurora-js-using-custom-templates), [Sample Scripts](./t-aurora-js-sample-scripts), and [Debugging, Advanced Techniques, and Sample Scripts](./t-aurora-js-sample-scripts-advanced-techniques-and-debugging).
7. Capture operational and support essentials.
   - Read [Service Information](./c-service-information), [Limitations](./t-aurora-js-limitations), and [Obtaining the Device MAC Address](./g-aurora-js-obtaining-the-device-mac-address).
8. Complete licensing and release-readiness checks.
   - Read the [Licensing Overview](../licensing) and [Release Notes Index](../release-notes), then review the version-specific release notes for your deployed Aurora Focus version.
