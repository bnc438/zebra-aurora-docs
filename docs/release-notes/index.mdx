# Release Notes

Use this section to track feature updates, bug fixes, and other changes by release.

## How to add a new release note

1. Copy `docs/release-notes/_release-notes-template.mdx` to a new `.md` or `.mdx` file in this folder.
2. Name it with a sortable release identifier where newer releases sort higher (example: `2026-03-01-aurora-focus-10-0-123.mdx`).
3. Replace all placeholder values (`[Product Name]`, `[Version]`, etc.) and publish.

Release notes are shown in descending file name order, so the most recent release appears at the top.
