## Getting Started in Aurora Focus

Follow this sequence when you are new to scripting and deployment in Aurora Focus:

1. Learn the guide layout and conventions.
   - Read [JavaScript Developers Guide](/docs/javascript-developers-guide), [About this Guide](../js-guide/t-aurora-js-about-this-guide), [Notational Conventions](../js-guide/r-notational-conventions), and [Icon Conventions](../js-guide/r-icon-conventions).
2. Confirm the runtime and editor context.
   - Read [Using the JavaScript Environment](../js-guide/c-aurora-js-environment), [Using the JavaScript Editor](../js-guide/t-aurora-js-using-the-javascript-editor), and [Using the Editor for a Deployed Job](../js-guide/t-aurora-js-using-the-editor-for-a-deployed-job).
3. Understand execution behavior before writing logic.
   - Read [Operation Mode](../js-guide/t-aurora-js-operation-mode), [Program Structure](../js-guide/c-aurora-js-program-structure), [Script Invocation](../js-guide/t-aurora-js-script-invocation), [Example Script Lifecycle](../js-guide/c-aurora-js-example-script-lifecycle), and [Managing Execution Time](../js-guide/t-aurora-js-managing-execution-time).
4. Build your first script around data flow.
   - Read [Input Data](../js-guide/t-aurora-js-input-data), [Constant Data](../js-guide/t-aurora-js-constant-data), [Output Data](../js-guide/t-aurora-js-output-data), [Job Status](../js-guide/t-aurora-scripting-job-status), and [Text Encoding](../js-guide/c-aurora-js-text-encoding).
5. Add device and interface controls as needed.
   - Read [TCPIP, RS232, and USB CDC Serial](../js-guide/t-aurora-scripting-tcpip-rs323-and-usb-cdc-serial), [HID Keyboard](../js-guide/t-aurora-js-hid-keyboard), [FTP Control](../js-guide/t-aurora-js-ftp-control), [GPIO Ports](../js-guide/t-aurora-focus-js-gpio-ports), and [Beeper](../js-guide/t-aurora-focus-js-beeper).
6. Validate and harden your implementation.
   - Read [Debugging](../js-guide/c-aurora-js-debugging), [Using Simulated Data](../js-guide/t-aurora-js-using-simulated-data), [Using Custom Templates](../js-guide/t-aurora-js-using-custom-templates), [Sample Scripts](../js-guide/t-aurora-js-sample-scripts), and [Debugging, Advanced Techniques, and Sample Scripts](../js-guide/t-aurora-js-sample-scripts-advanced-techniques-and-debugging).
7. Capture operational and support essentials.
   - Read [Service Information](../js-guide/c-service-information), [Limitations](../js-guide/t-aurora-js-limitations), and [Obtaining the Device MAC Address](../js-guide/g-aurora-js-obtaining-the-device-mac-address).
8. Complete licensing and release-readiness checks.
   - Read the [Licensing Overview](../licensing) and [Release Notes Index](../release-notes), then review the version-specific release notes for your deployed Aurora Focus version.
