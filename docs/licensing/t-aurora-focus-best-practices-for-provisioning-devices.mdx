Review the following best practices when provisioning a device to a license.

- Activate only the required number of seats that you want to use for provisioning devices.
- Do not hold on to claimed seats in Aurora Focus longer than needed.
    - This enables another user to use them to provision other devices.
- Syncronize provisioned seats to the license server frequently to ensure you are utilizing the active licenses on the device.
