# Deactivating a License Using the Web HMI

Use the Web HMI to deactivate a license on the device while connected to the internet.

:::note
Only provisioned licenses can be released.
:::

1. Click ![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-hmi-release-ico.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg) to release the license back to the license server.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-hmi-release-license.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

2. Observe that the license is removed from the list of **Active Licenses**.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-focus-licensing-hmi-remove.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)
