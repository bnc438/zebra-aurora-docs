# License Types and Toolsets

The following table outlines available licenses and their corresponding part number. Each license is associated with a toolset.

| Part Number | Web HMI Name | Description | Toolset | Applicable Devices |
| --- | --- | --- | --- | --- |
| LIC-SEN001-0100 | Sensor Toolset (Machine Vision) | Upgrade to Sensor Toolset. | MV Sensor | FS10, FS20, FS40, FS42, FS70, FS80 |
| LIC-SEST01-0100 | Standard Toolset (Machine Vision) | Fixed Industrial Scanner toolset upgrade from Sensor to MV toolset. | MV Standard | VS20, VS40/70 |
| LIC-STD001-0100 | Standard Toolset (Machine Vision) | Fixed Industrial Scanner upgrade to the Standard MV toolset (includes DPM and Fast 2D). | MV Standard | FS10/20, FS40/70, FS42, FS80 |
| LIC-DPM001-0200 | Fast 1D/2D Decoding with DPM Full | Full DPM (1D/2D/DPM), 60 frames per second. Enables Datacode decoding libraries for DPM reading and enhanced 2D. | Datacode | FS10/20, FS40/70, FS42, FS80; VS20, VS40/70 |
| LIC-2DF001-0200 | Fast 1D/2D Decoding | Fast 1D/2D, 60 frames per second (DPM not included). | None | FS10/20, FS40/70, FS42, FS80; VS20, VS40/70 |
| LIC-OCR002-0100 | Deep Learning Based OCR | Deep Learning OCR for VS devices. | Deep Learning OCR and MV Essential | VS20, VS40/70, NS42 |
| LIC-OCR003-0100 | Deep Learning Based OCR | Deep Learning OCR for FS devices. | Deep Learning OCR and MV Essential | FS40/70, FS42, FS80 |
| LIC-AD002-0100 | Deep Learning Based Anomaly Detection | Deep Learning based Anomaly Detection upgrade license for NS42 devices. | Deep Learning Anomaly Detection and MV Essential | NS42 |
| LIC-AD003-0100 | Deep Learning Based Anomaly Detection | Deep Learning based Anomaly Detection upgrade license for FS42 devices. | Deep Learning Anomaly Detection and MV Essential | FS42 |
| LIC-10LF-0000 | Connectivity Gateway Lite | Gateway Connectivity Lite, for FS10 devices. | None | FS10 |
| LIC-20LF-0000 | Connectivity Gateway Lite | Gateway Connectivity Lite, for FS20 devices. | None | FS20 |
| LIC-47LF-0000 | Connectivity Gateway Lite | Gateway Connectivity Lite, for FS40/FS70 smart cameras. | None | FS40, FS42, FS70 (pre-enabled on FS80) |
| LIC-EXLF-0000 | Connectivity Gateway Full | Gateway Connectivity Full, for FS and VS devices. | None | FS40, FS42, FS70, FS80, VS40, VS70 |
