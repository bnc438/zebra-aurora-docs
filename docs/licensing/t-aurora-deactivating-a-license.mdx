# Deactivating a License

Deactivate a license using Aurora Focus.

1. Click ![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-device-menu-ico.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg) and select **Deactivate** from the menu.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-focus-license-deactivate.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

2. Select the number of seats to deactivate.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-deactivate-seats.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

3. Click **Sync Now** to view the updated list of license keys.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-license-list-empty.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)
