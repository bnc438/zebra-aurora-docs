# Activating a License Using the Web HMI

Apply a device license, view available licenses, and deactivate licenses online using the Aurora Focus Web HMI.

1. Open the Aurora Focus Web HMI by entering the device's IP address into a web browser or clicking the IP address link under **View Devices** in Aurora Focus.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-focus-view-devices.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

2. If you are logged in as an Operator, click ![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-focus-licensing-profile-ico.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg) to log in to the HMI as an Admin.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-hmi-operator-nav.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

3. Use the default credentials for an Admin account:

- Username: admin
- Password: admin

4. After you log in as an Admin, click ![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-focus-licensing-gear-ico.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg) to access the settings menu.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-hmi-admin-nav.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

5. Select the Licensing tab.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-hmi-list.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

6. Ensure the following URL is present in the Licensing Server URL form field:

https://zebratechnologies.prod.sentinelcloud.com/ems/api/

Enter the **Product Key ID (PKID)** provided in the email when the license is procured.

7. Click **Activate License**.

8. The Active Licenses provides the following information:

- **License Index:** displays the license number.

- **License Description:** displays the license's feature or toolset name, for example, Connectivity Gateway Full or FS1D/2D Decoding.

- **License Type:** indicates the type of license and system version on which it was acquired. For example, Fast 1D/2D Decoding with DPM Full.

    - **Upgrade License (Legacy)** - an upgrade license activated using the legacy system. For example, Advanced Toolset (Machine Vision).

    - **Upgrade License** - an upgrade license using the current system. This license type can be provisioned in Aurora Focus and released in the Web HMI.

- **Activation Type** - licenses applied using the current system are provisioned.

- **Release:** Click ![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-hmi-release-ico.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg) to release the license back to the device's license server.
