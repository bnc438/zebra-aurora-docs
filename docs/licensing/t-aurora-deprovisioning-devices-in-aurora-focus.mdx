# Deprovisioning a License from a Device

Devices can be deprovisioned using Aurora Focus and released back to the license server as needed.

1. Go to **View Devices** and ensure that the device is managed by clicking **Manage**.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-manage-devices.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

2. Go to the **Licensing** tab in **Device Settings** and click **Enable Edit Mode**.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-enable-edit-mode.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

3. Click ![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-focus-licensing-trash-ico.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg) to release the license back to the server.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-remove-from-device.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)

4. Close the **Device Settings** tab and unmanage the device.

5. Return to the **Device Licensing** tab to confirm that the deprovisioned license was released and returned to the pool of claimed seats.

![](https://www.zebra.com/content/dam/techpubs/media/machine-vision/aurora-focus/g-aurora-licensing-remove-device-settings.png/_jcr_content/renditions/cq5dam.web.1280.1280.jpeg)
