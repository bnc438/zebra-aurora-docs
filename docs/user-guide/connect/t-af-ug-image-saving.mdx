Use Image Saving to save reads, no reads, and intermediate session images.
In the
Sessions to Save
section, specify if the images should be saved for pass, failed, or both job runs.
Sessions to Save
is useful to understand the final outcome of a job. If multiple frames are processed for a single job, enable
Save Immediate Session Images
. Saving partial image sequences can be helpful for further analysis or troubleshooting.
Enable
Save Result JSON
to save JSON files for each saved inspection result.
