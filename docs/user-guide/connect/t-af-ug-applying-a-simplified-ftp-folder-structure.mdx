If the default folder structure is not necessary, disable the default folder structure and apply a simplified structure.

Enable **Do not add pathname suffix** in the **Settings** dialog for the FTP connection.
To save pass or fail images only and avoid the JSON file, use the **Connect** chevron and deselect the **Enable JSON** checkbox.

The resulting folder structure is simplified, containing only the image files with their file name prefix indicating pass or fail status. Use **File Name Prefix** to separate Good Reads from No Reads without using the folder structure.
