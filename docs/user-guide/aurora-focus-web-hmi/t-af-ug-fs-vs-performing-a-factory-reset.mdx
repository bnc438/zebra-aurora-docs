A factory reset deletes all created jobs on the camera.
A factory reset deletes all created jobs on the camera. It is critical to save all jobs and user-specific information before performing the factory reset.
Click **Operator** to launch the login window.
Enter the following credentials:
User ID: admin
Password: admin
Click **Login**.
Click the **Settings** icon.
On the **General** tab, click **Reset All** in the **Factory Reset** field, and wait for the device to reboot (approximately 45-60 seconds).
