When logged in to the Zebra Web HMI, the application presents a dashboard with key hardware metrics such as average inspection per minute, total uptime, temperature, CPU load, communication status, average pass/fail, and resource utilization.

Use Google Chrome for optimal performance while using the Web HMI.
