The Brightness tool calculates the average pixel brightness value in a Region of Interest (ROI).




## General Brightness Settings

Settings
Description
Fixture
Determine the fixture for the ROI shape.
Image Type
Select which image this inspection tool should use.
Value Range
Set the minimum and maximum values.
Histogram
Shows the number of pixels at each value.
