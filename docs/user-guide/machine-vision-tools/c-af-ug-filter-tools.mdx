Filter Tools are facilitated by the use of a kernel. A kernel is repeatedly centered at each pixel within the dimensions of the region that is being transformed. Every pixel is either added to the resulting region or not, depending on the operation-specific condition set on the minimum number of kernel pixels that have to overlap with actual input region pixels in the given position of the kernel.
