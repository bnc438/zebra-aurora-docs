A blob is a set of connected light-or dark-colored pixels. The Locate Blob tool detects blobs that pass specific filter parameters within a region of interest.

This filter is helpful for quickly segmenting an image. The Locate Blob tool performs a series of operations on the image, including; thresholding using the
Threshold
parameter and removing holes by setting
Fill Holes
.

Filter Boundary blobs by setting
Allow Boundary
blobs.
