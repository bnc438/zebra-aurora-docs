The Circle Diameter tool measures the diameter of a circle located in the Region of Interest (ROI).

## Circle Diameter Settings

| Settings | Description |
| --- | --- |
| Find By | Define the approach used to search for the circle: - Best Score - Largest Circle - Smallest Circle |
| Transition Type | Select the type of edge transition used to find the circle: - Both - Blended - Dark to Light - Light to Dark |
| Both Blended Minimum Edge Magnitude | Sets the minimum contrast value used to define the edge of the circle. |
| Edge Profile | Displays the contrast score of the features across the Region of Interest (ROI). |
| Diameter Range | The Diameter range parameter defines the diameter value for which the tool returns a pass or fail status. |
| Scale Calibration | Calibrates pixel values to engineering units. |
