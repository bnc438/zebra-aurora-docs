When using the
Image Type
drop-down menu to use a filter or tool output as the source image, the ROI of the tool must be fully contained within the ROI of the source images. If the tool's ROI exceeds the source's ROI, the tool automatically issues a failing result.

In the following image, the
Brightness
tool uses the
Binarize
tools output as the source image. As a result, the ROI of the
Brightness
tool must be fully contained within the
Binarize
tools source ROI. If there is an overlap between the two ROIs, the
Brightness
tool issues a fail.
