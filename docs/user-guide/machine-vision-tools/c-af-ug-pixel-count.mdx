The Pixel Count tool provides the number of pixels of a user-specified value (0-255) within a Region of Interest (ROI).

## General Pixel Count Settings

| Setting | Description |
| --- | --- |
| Threshold | The minimum and maximum threshold values for Pixel count. |
| Pixel Count | The minimum and maximum pixel count values are considered for passing. |
| Histogram | Shows the number of pixels at each value. |
