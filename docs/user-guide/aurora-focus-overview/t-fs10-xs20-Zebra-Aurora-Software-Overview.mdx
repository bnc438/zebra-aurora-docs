Zebra Aurora Focus is a unified platform with an intuitive interface for setting up, deploying, and running Fixed Industrial Scanning or Vision System jobs to control enterprise-wide manufacturing and logistics automation solutions. This tool also can scale in support of new codes and increase scanning speed with the potential to upgrade to machine vision functionality via software license upgrade.

## Key Capabilities

- **Fixed Industrial Scanning (FIS)** — Configure barcode reading jobs with support for 1D, 2D, DPM, and Datacode symbologies. Tune decoder parameters, set up triggers, and deploy jobs to connected devices.
- **Machine Vision (MV)** — Build and deploy machine vision jobs using tools for identification, locate, measurement, counting, presence/absence, and filtering. Stack multiple tools onto a FlowBuilder workflow.
- **Connectivity Gateway** — Manage leader/follower device topologies, PLC integration, GPIO triggering, Modbus communication, and ZETI command interfaces through a centralized gateway.
- **JavaScript Scripting** — Customize output formatting and data processing using the built-in JavaScript editor with IntelliSense, templates, and Monaco Editor integration.
- **Image Management** — Capture and analyze images using Image Perfect+ (up to 16 unique acquisition settings), sensor cropping, and filmstrip controls.
- **Device Management** — Discover, configure, back up, restore, and update firmware on connected devices over USB or Ethernet.

## Supported Devices

Aurora Focus supports the following Zebra fixed industrial scanners and smart cameras:

- FS10, FS20 — Fixed Industrial Scanners
- FS40, FS42 — Smart Cameras with Machine Vision capability
- NS42 — Network Smart Camera

## Getting Started

To begin using Aurora Focus, review the [Minimum PC Requirements](../getting-started/c-aurora-minimum-pc-requirements) and install the application. Connect a device, create a job, and deploy it to the device using the guided workflow in the application.
