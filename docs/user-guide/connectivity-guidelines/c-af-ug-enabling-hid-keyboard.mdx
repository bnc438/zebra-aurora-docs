Checking this setting will enable USB-HID output. Job result/output will be streamed over a USB connection to the host PC as if the device was a keyboard. The output streams into any field with focus, such as notepad, excel, or other host-based apps.


When this setting is applied, the device will automatically reboot. If USB is your sole way of communicating with the device (as opposed to via ethernet cable) and you have a job that automatically starts on startup, it may be difficult to communicate with the device if the job is running and sending output over the USB simultaneously. Ensure that you have de-selected the other options before enabling HID.
