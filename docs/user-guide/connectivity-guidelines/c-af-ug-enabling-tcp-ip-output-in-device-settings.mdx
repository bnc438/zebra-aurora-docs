TCP/IP output is disabled by default and must be explicitly enabled in the device's settings.


1. Select the representative communication type tab. For example, Ethernet Port 1.
2. Scroll to TCPIP Settings.
3. Check the Enable TCP/IP Control checkbox (disabled by default).
