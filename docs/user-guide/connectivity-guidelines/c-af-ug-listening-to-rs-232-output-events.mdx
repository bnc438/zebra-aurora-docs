1. Open an RS-232 client terminal software such as PuTTy.
2. Establish a connection to the device by clicking **Session**.
3. Set **Serial Line** to the COM Port Number.
4. Set **Speed** to the Serial speed defined in Aurora Focus.
5. Click **Open** to open a serial connection in terminal.
6. If the connection is successful, observe the terminal window with a green cursor.
7. Trigger the job and observe the results in the terminal window to confirm that the device is configured as expected.
