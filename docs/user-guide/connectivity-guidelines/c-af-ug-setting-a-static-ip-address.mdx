DHCP is the default mode for IP addressing.

1. Disable the **Enable DHCP** checkbox.
2. Change the network-related settings based on your network configuration. For example:
   1. Open a command prompt.
   2. Type the command: ipconfig
   3. Identify a valid network interface. All TCP/IP network configuration values display on the host PC.
   4. In Aurora Focus, enter the following values:
      - IP Address: 192.168.4.xxx (where xxx is any value from 1-255)
      - Subnet Mask: 255.255.252.0
      - Default Gateway: 192.168.4.1
3. Click **Apply** when complete. The device reboots with the new static IP address.
