:::note
RS-232 output is disabled by default and must be explicitly enabled in Device Settings.
:::

1. Select the associated communication type tab.
2. Scroll down to **RS-232 Settings**.
3. Ensure that the **Enable RS-232 Results** checkbox is checked.
4. Click **Apply** within the **RS-232 Settings** area.

:::note
Record the **Speed, Data Bits, Parity, Stop Bits** — you may need these settings later to listen for RS-232/Serial results from external software.
:::
