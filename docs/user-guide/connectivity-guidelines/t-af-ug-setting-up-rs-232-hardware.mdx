The following accessories are necessary for a serial interface connection:
- Power/GPIO Flying Leads cable
- USB to Serial Adaptor
- DB9 RS-232 D-SUB Serial Adaptor
Connect the following wires from the flying leads cables to the DB9 Serial adaptor.
## Flying Leads to DB9 Serial Adapter Wire Diagram

| Pin | Color | Description |
|-----|-------|-------------|
| 1 | Yellow | OUT1 |
| 2 | White/Yellow | TXD |
| 3 | Brown | RXD |
| 4 | White/Brown | IN1 |
| 5 | Violet | RTS |
| 6 | White/Violet | COMMON_IN |
| 7 | Red | DC_IN |
| 8 | Black | GND |
| 9 | Green | COMMON_OUT |
| 10 | Orange | IN0 |
| 11 | Blue | OUT0 |
| 12 | Grey | CTS |
| SHELL | Bare | SHIELD |

1. Connect DB9 Serial Adapter (female) to USB-to-Serial adapter (Male).
2. Connect USB-to-Serial adapter USB end into host PC.
3. Identify the COM port associated with the USB-to-Serial adaptor using Device Manager on a Windows PC.
4. Expand the Ports field and identify the USB-to-Serial adaptor.
