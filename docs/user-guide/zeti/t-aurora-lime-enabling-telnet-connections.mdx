Use a Telnet connection with a PC-based terminal over ZETI with a device.
Press Win + R to open Run.
Search for the Control Panel and click
OK
.
Search for the Control Panel and click
Programs and Features
Using the left panel, click
Turn Windows Features On or Off
.
Enable telnet client Windows features dialog.
In the Windows Features dialog, scroll down and select
Telnet Client
.
You can also use a terminal client such as Teraterm or Putty. The following image displays the procedure using Putty.



After opening the session, type the command name and press
Enter
.
