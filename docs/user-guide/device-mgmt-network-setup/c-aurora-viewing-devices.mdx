Click **View Devices** to access additional device information such as name, part number, IP, serial number, firmware version, and status.

Click the device IP address to access the Zebra Web HMI.

## Device Information

| UI Element | Description |
|---|---|
| Backup Device | Create a backup of the selected device. The resulting file is an encrypted archive that can be used only by the restore function. |
| Restore Device | Select a backup file and restore the device configuration. The process also triggers a complete reboot of the device. Applying a backup from one model type to another can create some issues. |
| Update Firmware | Use this command to update the device firmware. This operation can also be done using the Web HMI. |
| Download Logs | This command refreshes the list of discoverable devices on the network. |
| Zebra Connectivity Gateway HMI | Access the Zebra Connectivity Gateway HMI. |
| Reboot | Reboot the device. |
| Factory Reset | Perform a factory reset on the device. |

Click on the device row to view additional device information.
Copy the device's Serial Number, Part Number, and Firmware Version before contacting technical support.

## Device Information

| UI Element | Description |
|---|---|
| Overview | View device information, including firmware version, serial number, manufacturing date, core services version, and ChiCore library version. |
| Communications | View communication settings, including connection type and IP mode. |
| System | View system settings, including DHCP timeouts, uptime, sensor type, and firmware version, and focus calibration type. |
| Status | View lens and illumination settings, including lens type, illumination type, external illumination, external illumination GPIO, and number of available GPIOs. |
