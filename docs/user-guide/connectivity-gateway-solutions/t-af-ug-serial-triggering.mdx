Set up a Serial Trigger from the host to the Connectivity Gateway.
There is no serial connection between the leader and the follower devices; triggering between the devices occurs over TCP/IP.
Navigate to the
Configuration
screen using the left menu on the Home screen.
Locate the
Serial
section and click
Config
to set the serial settings for the device.
Configure the form fields and click
Confirm
to enable the settings on the device.
