Use the Connectivity Gateway to send results to the connected PLC device.
Navigate to the **Admin Settings** screen from the left menu on the **Home** screen and click **PLC Config**.
Click **Results Tag** on the **PLC Configuration** screen.
Select the PLC device from the menu and enter a value for the tag.
On the **PLC Configuration** screen, toggle **Output via PLC** to **On** (green).
The results tag supports a string of up to 4096 characters, including the CR+LF terminator.
After the configuration is complete, navigate to the **Control PLC** screen, select the PLC device from the menu, and click **Start** to begin sending results.
