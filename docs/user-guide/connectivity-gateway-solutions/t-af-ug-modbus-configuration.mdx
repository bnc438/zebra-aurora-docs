Configure a device to use the Modbus communication standard.
Navigate to the
Admin Settings
menu from the
Home
screen and click
Modbus Config
.
Slide the toggle to the right to enable
Modbus Triggering
or
Output to Modbus
.
Click
Control Device
to access controls to the Modbus device.
Select the Modbus device from the menu and click
Start
to begin controlling the device. Click
Stop
to end the session or
Delete
to remove the device. When you are finished controlling the device, click
Back
to return to the
Admin Settings
menu.
Click
Results
to access Modbus results and configure where results are sent to.
Select the Modbus device from the menu and define a
Holding Register
.
Click
Confirm
to save changes and return to the
Admin Settings
menu.
Click
Create Device
to configure the device information.
Provide the required device information and click
Create
to return to the
Admin Settings
menu.
Click
Monitoring
to monitor a specific coil on the Modbus device
Select a Modbus device from the menu and determine a coil number to monitor.
