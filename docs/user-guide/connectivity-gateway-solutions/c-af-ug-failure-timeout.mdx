Set a failure timeout to account for trigger input delays.

1. Navigate to the **Configuration** screen using the left menu on the **Home** screen.
2. Use the bottom left **Failure Timeout** section to specify a timeout.
3. Click **Update** for the changes to take effect.

Ensure your failure timeout accounts for any trigger input delays specified on the follower side.
