1. Create a new job for the follower device and navigate to the Build tab in Zebra Aurora.

2. Navigate to the **Symbologies** section. Ensure that only the symbologies necessary for the application are enabled.

3. Navigate to the **ManyCode** tab. Configure the total number of Barcodes to Decode (3 max). Sort by symbology and enable partial results.

4. Navigate to the **Data Formatting** tab and select **Advanced Formatting**.

5. Select **ManyCode** and input the following:

   - Prefix: None
   - Data: Many Code String
   - Custom String: "~~$$~~"
   - Suffix: None
   - Delimiter
     - Data Type: None
     - End: None
     - ManyCode String: None

6. Select **All Symbologies** and input the following:

   - Prefix: None
   - Data: Symbology, Custom String ","
   - Full String: Base 64
   - Custom String: "~~##~~"
   - Suffix: None
   - Delimiter
     - Data Type: None
     - End: None
