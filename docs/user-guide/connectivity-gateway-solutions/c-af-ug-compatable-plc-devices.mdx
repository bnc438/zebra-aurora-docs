The Connectivity Gateway supports communication to and from the following PLC models.

| PLC Manufacturer | Supported Models/Protocols |
|---|---|
| Siemens | S7 TCP/IP, Logo!, S7-200, S7-300, S7-400, S7-1200, S7-300 (ERPC), ET 200 Pro, S7-1500, SINUMERIK 840D, PCS 7 |
| Rockwell | ControlLogix CPU, CompactLogix CPU, Micro800 CPU |
| Mitsubishi | QCPU (Built-in Ethernet), LCPU (Built-in Ethernet), RCPU (Built-in Ethernet), FX3CPU, FX5CPU, GOT, ACPU |
| Omron | CS1 CPU, CJ1 CPU, CV1 CPU, NJ CPU, CP1 CPU, NX CPU, CJ2 CPU, NE1S CPU |
| Modbus | TCP/IP |
