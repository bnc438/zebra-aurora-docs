Create a PLC device to connect to the Connectivity Gateway.

1. Select **PLC Config** from the **Admin Settings** menu.
2. Select **Create PLC** from the **PLC Configuration** menu.
3. On the Create PLC screen, select the **Manufacturer** and **Model** of the PLC, specify a **Friendly Name** for the PLC, and enter the **IP Address**.
4. Click **Confirm** and observe the confirmation message.
