Before setting up TCP/IP triggering on the device, configure the Host to Leader port and follower devices.
Navigate to the
Configuration
screen using the left menu on the
Home
screen.
Locate the
Trigger Strings
section and click
Config
.
Select the terminator for each trigger from the drop-down menu and click
Update
to apply the changes.
Serial Interface does not support a null terminator.
