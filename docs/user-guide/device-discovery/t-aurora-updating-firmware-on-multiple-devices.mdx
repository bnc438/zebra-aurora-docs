list to perform a firmware update.

Select a firmware file from the File Explorer to use the

option.

Use
Dual Update
to update the primary device partition, reboot the device, update the secondary partition, and reboot the device again.

Use
Force Update
to apply the same firmware version to all devices.

Firmware updates are performed on Connected devices only.
It is not possible to open more than one
Update Firmware
window.
