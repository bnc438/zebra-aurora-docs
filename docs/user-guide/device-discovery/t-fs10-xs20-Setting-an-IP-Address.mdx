Connect to a camera by manually entering its IP address and clicking **Add Via IP Address** in the bottom right corner of the **View Devices** screen. Use this feature for devices with a known IP Address that are not automatically discoverable in Zebra Aurora Focus. You can also connect via IP address by entering the IP address (or hostname) into the **Add New Device via IP Address** form field and clicking **Connect**.
