Click
Setup New Device
on the home screen to configure an FS or VS Emulator.



Select Emulator Selection and configure the settings for your emulator from drop-down menu.


## Setup New Device

Setting
Description
Type
Select the toolset type for the emulator.
Resolution
Select the desired resolution of the emulator.
Model
Select a device model of the emulator.
License
Select the license type for the emulator.
