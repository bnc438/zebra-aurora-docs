If there is no communication between the smart camera and the FTP Server application, add the FTP server application to the Windows Firewall Rules using the following procedure.
Open Windows Defender Firewall.
Click
Allow an app or feature ….
Click
Allow another app…
.
Select the executable of the FTP server.
Select all networks.
Click
OK
to apply the settings.
