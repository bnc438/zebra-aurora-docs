If there is no communication between the smart camera and the FTP Server application, add the FTP server application to the Windows Firewall Rules using the following procedure.

1. Open Windows Defender Firewall.
2. Click **Allow an app or feature…**.
3. Click **Allow another app…**.
4. Select the executable of the FTP server.
5. Select all networks.
6. Click **OK** to apply the settings.
