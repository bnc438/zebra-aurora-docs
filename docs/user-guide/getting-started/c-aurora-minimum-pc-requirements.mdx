## Minimum Requirements

| Item | Description |
| --- | --- |
| CPU | Intel or AMD 64-bit processor |
| Random Access Memory | 4 GB (Recommended: 8GB) |
| GPU | GTX 1030, Intel UHD 620 or equivalent |
| Graphics Memory | 2 GB (Recommended: 4GB) |
| Disk Space | 4 GB free disk space |
| Monitor Resolution | 1280 x 800 (Recommended: 1920 x 1080) |
| Operating System | Windows 10 or higher |
