The Jobs Toolstrip provides access to available devices or emulators to apply the job to, the barcode type, and the detected power source. Save the configuration by clicking the disk icon.




## Jobs Toolstrip

Setting
Description
Device/Emulator Selector



Select the device or emulator for the job to be deployed to.
Barcode Type



Select the applicable barcode type for the job.
Power Source



Displays the power source type that the device is connected to.
Save Options



Saves the job configuration.
