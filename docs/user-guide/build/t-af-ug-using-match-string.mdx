Use Match String to determine if the code has a specific string present.
Match String supports the following tools:
- Datacode
- Deep Learning OCR
- Read Barcode
Datacode
Deep Learning OCR
Read Barcode

For cases where match string functionality is not necessary, ensure
Barcode String Match
is disabled.



In cases where the data code contains the given string, the job passes.



In cases where the data code does not contain the given string, the job fails.
