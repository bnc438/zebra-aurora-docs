The **Build** tab facilitates job configuration and deployment for FIS and MV tools.

## Build Settings

| Settings | Description |
| --- | --- |
| View Results | View the results of recent jobs. |
| Image Viewer | Observe the tool's analysis of the image. |
| Filmstrip | View the series of images captured by the device or upload a previously captured set of images. |
| Deploy | Run the job as configured in FlowBuilder. |
