Use fixturing tools to focus on a specific symbology in environments where codes are processed rapidly and may be presented in different orientations.


All 1D and 2D symbologies are compatible with fixturing tools.
Create a new job.
Add the Read Barcode tool to the FlowBuilder.
Add an additional tool, such as Deep Learning OCR.
Ensure that Manycode is disabled before using fixturing.
Select the first tool (Read Barcode) from the Fixture drop-down list.
Deploy the job and move the test image. Observe the Deep Learning OCR tool's ROI following the Barcode tool results.
