Match Strings are configurable in Aurora Focus for Datacode, Deep Learning OCR, and Read Barcode tools.
Add the tool in
FlowBuilder
using the
Build
tab to view match string options.
Select the
Barcode String Match
mode from the drop-down menu.
- Disabled (Default) - no match string is used, and every string is accepted.
- Exact string - the result must match the exact string provided.
- String Contains - the result must contain the given string.
- Regex - result acceptance follows the given regex rule.

Barcode String Match
mode is disabled by default and the match string value is empty. In this case, the tool accepts all read codes.
Click
Last Decode
to use the last successful read string as the current match string value.
