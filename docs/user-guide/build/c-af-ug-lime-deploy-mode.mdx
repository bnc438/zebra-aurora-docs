Use Deploy mode to view the job results and decode summary.

Click **Deploy** while in **Edit** mode
to enter **Deploy** mode
and view job results and decode summary for read count, total pass/fail, tool time, and quality information.
The Filmstrip is not available while in Deploy mode.
