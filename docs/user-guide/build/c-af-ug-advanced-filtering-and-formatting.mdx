The following functionalities are not supported in Aurora Focus:
- Non-standard, customer-specific check digit calculations
- Location based Pattern Match requirements based on barcode positional dependencies such as Code 128 is on the left of a UPC
- Interjection of time delays between transmitted barcode data
- Non-ASCII values
