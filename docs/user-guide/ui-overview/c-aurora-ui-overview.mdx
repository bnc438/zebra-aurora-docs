The **Getting Started** screen features functionality such as viewing and configuring connected or emulated devices and settings for Fixed Industrial or Machine Vision scanning jobs.

## Zebra Aurora Focus Home

| Setting | Description |
| --- | --- |
| Setup New Device | Provides access to the configuration of existing (previously connected) and virtual (emulated) device settings and program jobs. |
| View Devices | View connected devices on your network. |
| Settings | Click **Settings** to manage localization and notification settings or clear the cache. Enable Automatic Fixturing options and **Image Capture Disk Management** settings on the **Job Management** tab. Click the drop-down menu to sort by: - Recent Activities - Recent Jobs - Recent Devices Click the filter icon to filter by: - Type - Name - Model Name - IP - Last Accessed |
